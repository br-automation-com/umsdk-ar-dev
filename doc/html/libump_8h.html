)

An advanced API for reading positions of certain manipulator and allowing to control the position value timings.

A zero time_limit (LIBUMP_TIMELIMIT_CACHE_ONLY) reads cached positions without sending any request to the manipulator. Value -1 (LIBUMP_TIMELIMIT_DISABLED) obtains the position always from the manipulator.

Parameters
hndlPointer to session handle
devDevice ID
time_limitTimelimit of cache values. If 0 then cached positions are used always. If
[out]xPointer to an allocated buffer for x-actuator position
[out]yPointer to an allocated buffer for y-actuator position
[out]zPointer to an allocated buffer for z-actuator position
[out]wPointer to an allocated buffer for w-actuator position
[out]elapsedPointer to an allocated buffer for value indicating position value age in ms
Returns
Negative value if an error occured. Zero or positive value otherwise
