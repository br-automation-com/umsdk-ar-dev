For C++ application there is an inline wrapper class LibUm

To the SDK under C application (or any accepting a library with a standard C API) examine the C API defined by libum.h.

First open the port by calling LibUm::open or um_open.

How to use the methods or functions examine the provided examples.

Before terminating the program call LibUm::close or let the object's destructor to call it, for C-API call um_close.
